# Personal Portfolio Website

A modern, responsive portfolio website built to showcase my projects, skills, and growth as a Frontend Developer.


## Live Demo

 Live Site: https://isinkaye-praise.vercel.app/  
 Repository: https://github.com/Bebe-dev/Isinkaye-Praise  


##  Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth animations using Framer Motion
- Project showcase section with:
  - Live demo links
  - GitHub repository links
- Downloadable CV
- Clean and minimal UI
- Structured and reusable components
- Modern layout with overlay effects


##  Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion


##  What I Learned

Building this portfolio helped me strengthen my understanding of:

- Responsive design using Tailwind CSS
- Absolute positioning and overlays
- Implementation of animation using Framer Motion
